# 报表模块


## 报表列表

### 接口介绍

>uri: /charts
>method: GET
>返回数据格式: JSON


### 参数列表
> proj_id 项目编号 Integer 0表示全部项目
> chart_id chart编号,逗号隔开,可同时获取多种数据  // 0-表示所有 1-收支, 2-平均人效, 3-人员流失率

### 调用示例
```
{
  "proj_id": 1,
  "chart_id": "1, 2, 3"
}
```
